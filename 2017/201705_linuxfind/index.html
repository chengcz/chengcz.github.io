find，Linux常用文件搜索命令简要整理

基本用法

find基本用法如：find [path] <search regular> [action]

  • 可选参数path，支持空格分隔的多路径查找，默认为当前路径
  • 可选参数action
    • -print，匹配文件输出到stdout，默认
    • -exec，对find匹配到的文件执行该参数所给出的shell命令，格式为command { } \;，其中{}表示匹配文件
    • -ok，于-exec类似，执行前给出提示是否执行
